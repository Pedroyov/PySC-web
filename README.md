# Sitio web de Pasión y Sentimiento Cultural

Primera versión estática preparada para GitHub Pages.

## Páginas

- Inicio
- Nosotros
- Historia
- Proyectos
- Galería
- Contacto

Las páginas todavía no desarrolladas muestran un mensaje de “Estamos trabajando”.

## Publicar en GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube todos los archivos de esta carpeta.
3. Ve a Settings > Pages.
4. En Source selecciona `Deploy from a branch`.
5. Selecciona la rama `main` y la carpeta `/root`.
6. Guarda los cambios.

La página estará disponible en:

`https://TU-USUARIO.github.io/NOMBRE-DEL-REPOSITORIO/`
